# شمس المعارف الكبرى v19

**نظام روحاني متكامل مستخرج من مخطوطة شمس المعارف الكبرى للإمام أحمد بن علي البوني (توفي ٦٢٢هـ)**

---

## نظرة عامة

شمس المعارف v19 هو نظام برمجي شامل يُحوّل محتوى مخطوطة شمس المعارف الكبرى إلى تطبيق عملي يتضمن:

- **علم الحروف والأبجد** — حساب الجمل، تحليل الأسماء، استنطاق الحروف
- **الأوفاق السحرية** — توليد المربعات السحرية (3×3 حتى 10×10)
- **الجفر** — الجفر الصغير والمركب والجفر التاريخي
- **الزايرجة** — الزايرجة الخطية والدائرية والمركزية
- **علم الرمل** — توليد الأشكال الـ16 وتحليل اللوح الكامل
- **الكيمياء الروحانية** — ربط المعادن بالكواكب، مراحل الخيمياء
- **السيميا** — مقالات الخنفطريات الأحد عشر
- **المنازل القمرية** — حسابات فلكية دقيقة بـ NASA Ephemeris
- **نظام الاقتراح الذكي** — يقترح أفضل الأعمال بناءً على الوقت الحالي

---

## المتطلبات

- Python 3.9 أو أحدث
- Windows 10/11 (للـ Start.bat) أو Linux/macOS

---

## التثبيت والتشغيل

### Windows (الطريقة السهلة)
```
انقر نقراً مزدوجاً على Start.bat
```
سيقوم الملف تلقائياً بـ:
1. التحقق من Python
2. إنشاء البيئة الافتراضية
3. تثبيت المتطلبات
4. تشغيل الخادم
5. فتح المتصفح على http://localhost:5000

### Linux/macOS (يدوي)
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

- يحلل الوقت الحالي (الساعة الكوكبية، المنزل القمري، اليوم)
- يقترح أفضل الأعمال للحظة الحالية
- يُحدد إذا كانت الساعة مباركة أم مذمومة
- يقترح البخور والأسماء الحسنى المناسبة
- إمكانية فحص أي نية لمعرفة أفضل وقتها

## نقاط API الرئيسية

### الحساب الأساسي
- `POST /analyze` — التحليل الشامل (الاسم + الأم + النية)
- `POST /api/process` — المعالجة الكاملة مع كل المحركات

### نظام الوقت الذكي (v19 جديد)
- `GET /api/time/suggest` — اقتراح الأعمال للوقت الحالي
- `POST /api/time/suggest_for` — فحص نية معينة

### الجفر
- `POST /api/jafr/simple` — الجفر الصغير
- `POST /api/jafr/compound` — الجفر المركب
- `POST /api/jafr/historical` — الجفر التاريخي (ملوك/فتن)
- `GET /api/jafr/table` — جداول الجفر

### الزايرجة
- `POST /api/zairja/ask` — سؤال الزايرجة
- `POST /api/zairja/center` — الزايرجة المركزية (v19 جديد)
- `POST /api/zairja/circular` — الزايرجة الدائرية
- `POST /api/zairja/full` — القراءة الكاملة

### الكيمياء (v19 جديد)
- `GET /api/alchemy/metal/<planet>` — معدن الكوكب
- `GET /api/alchemy/stages` — مراحل الكيمياء السبع
- `GET /api/alchemy/recipes` — وصفات الكيمياء
- `POST /api/alchemy/elixir` — حساب قوة الإكسير

### السيميا (v19 جديد)
- `GET /api/symia/articles` — مقالات الخنفطريات
- `GET /api/symia/article/<num>` — تفاصيل مقالة
- `POST /api/symia/ritual` — تنفيذ طقس
- `GET /api/symia/khunfutriyyat` — طقس الرماد الأسود

### المعرفة الموسعة (v19 جديد)
- `GET /api/v19/knowledge/letters` — الحروف الكاملة
- `GET /api/v19/knowledge/divine_names` — الأسماء الحسنى
- `GET /api/v19/knowledge/planets` — الكواكب
- `GET /api/v19/knowledge/zodiacs` — البروج
- `GET /api/v19/knowledge/thrones` — الأعراش
- `GET /api/v19/knowledge/gates` — الأبواب الأربعون

## استكشاف الأخطاء

**الخادم لا يبدأ:**
- تأكد من تثبيت Python 3.9+
- شغّل `pip install -r requirements.txt` يدوياً
- تحقق أن المنفذ 5000 غير مستخدم

**الحسابات الفلكية غير دقيقة:**
- تأكد من وجود ملف `de421.bsp` في المجلد
- إذا لم يكن موجوداً، يمكن تحميله تلقائياً عند أول تشغيل

**مفتاح الترخيص:**
- الإصدار التجريبي يعمل بدون مفتاح مع صلاحيات محدودة
- للنسخة الكاملة تواصل مع المطور
